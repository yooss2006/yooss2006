# 안녕하세요. 
프론트엔드 개발자 **유순상**입니다. 😀

학점은 3.97로 나름 성실하게 대학생활을 보냈다고 생각합니다. 

> 대학생활에서 배운 것은
> 
- 컴퓨터구조, 자료구조 등의 개념을 익혔습니다.
- 수업으로 포토샵을 배웠고 이에 흥미를 느껴 더 공부해 능숙하게 다룰 줄 압니다.
- 웹 프로그래밍 수업을 듣게 되어 웹 개발자에 흥미를 가지게 되었습니다.
