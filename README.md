학점은 3.99로 정말 아쉽게 4에 도달하지 못했지만 나름 성실하게 대학생활을 보냈다고 생각합니다. 

> 대학생활에서 배운 것은
> 
- 컴퓨터구조, 자료구조 등의 개념을 익혔습니다.
- 수업으로 포토샵, 일러스트, 프리미어프로와 같은 편집 기술을 배웠고 그중에 포토샵에 흥미를 느껴 더 공부해 능숙하게 다룰 줄 압니다.

- 웹 프로그래밍 수업을 듣게 되어 웹 개발자에 흥미를 가지게 되었습니다.
